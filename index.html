Jquery Repeter by: ELIBO. Ver código en Github

Descripción: jQuery Repeter, sencillo plugin que repite elementos del DOM mediante botones para "añadir nuevo" o "eliminar elemento", Perfecto para Formularios dinámicos que requieren múltiples conjuntos de inputs iguales.

Básico

  • Item Inicial
Javascript: $('.dinamic').repeter();

Inserción en otro elemento

  • Item Inical
Javascript: $('.dinamic-2').repeter({elements: {insertIn:'#otro'}});

Inserción con espejo y mismo template

  • Item Inicial
  • Item Inicial
Javascript: $('.dinamic-3').repeter({ elements:{ mirror:{selector:'#espejo'} } });

Inserción con espejo y otro template

  • Item Inicial
  • Item Inicial
Javascript: $('.dinamic-3').repeter({ elements:{ mirror:{selector:'#espejo', tplSelector:'.template-mirror'} } });
