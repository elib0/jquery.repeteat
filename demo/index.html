Jquery Repeter v2 by: ELIBO. Ver código en Github

Características:
  • Repite o elimina fragmentos HTML, Útil para formularios que requieren array inputs.
  • Tiene la posibilidad de usar animaciones CSS3 al eliminar item.(Recomendado: Animate.css)
  • Si es usado el Bootstrap Validator activa los inputs al agregar item y actualiza las validaciones.
  • Puede repetir template en otro contenedor.
  • Puede repetir en dos contenedores a la vez, con posibilidad de usar template alternativos para la insercion.(Espejo)
div.dinamic

  • Item Inicial

Inserción en otro elemento
(Con animacion css3 usando animated.css)

  • Item Inical
Javascript:
$('.dinamic-2').repeter({
  elements:{
    insertIn:'#otro',
    removeClasses:'animated bounceOutRight'
  }
});
